Sensor Car Upgrade

Concept 4.1 (Click the image to view fullscreen)

This project holds the next round of changes for the tracking car. The intended follow-up is better cable management, a cleaner chassis layout, and more controlled testing of how the sensors react on tighter turns.

Planned focus:

  • Control behavior: test line tracking on sharper curves and uneven lighting.

  • Status: outlining what should change in the board placement before a new chassis pass.

Modular Chassis

Concept 5.1 (Click the image to view fullscreen)

The modular chassis page is reserved for a build that compares laser-cut and printed parts within one design language. The idea is to make a chassis that can swap panels or test pieces without rebuilding the whole structure.

Planned focus:

  • Hybrid fabrication: combine printed geometry with flat-pack structural pieces.
  • Iteration speed: create removable sections to test parts one at a time.
  • Status: defining a shared scale and connection method before modeling starts.

Hand Refinement

Concept 6.1 (Click the image to view fullscreen)

Planned focus:

  • Mechanical revision: evaluate where the current geometry limits movement or durability.
  • Presentation: build a cleaner visual sequence for concept, print, and assembly stages.
  • Status: collecting measurements and deciding which joints deserve the next redesign effort.
